# Туристический гид

- Приложение использует фреймворк **MapKit** для работы с картами
- Оно также содержит список самых известных достопримечательностей Санкт-Петербурга
- Вы можете найти местоположение любой из них, нажав на картинку-превью
- Также возможно найти достопримечательность из списка, написав в поисковой строке её название или описание
